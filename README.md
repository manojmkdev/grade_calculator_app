# Grade Calculator App

A clean and simple web application that helps students calculate their **total marks**, **average**, and **final grade** across multiple subjects.

🔗 **Live Demo:** [https://manojmkdev.github.io/grade_calculator_app/](https://manojmkdev.github.io/grade_calculator_app/)

---

## ✨ Features

- ➕ Dynamically add or remove subjects
- 🧮 Calculates total marks, average score, and final grade instantly
- 📱 Responsive and user-friendly interface
- ⚡ Lightweight — no frameworks or dependencies required

---

## 🚀 Getting Started

### Prerequisites

All you need is a modern web browser. No installations required.

### Run Locally

1. **Clone the repository**
   ```bash
   git clone https://github.com/manojmkdev/grade_calculator_app.git
   ```

2. **Navigate to the project folder**
   ```bash
   cd grade_calculator_app
   ```

3. **Open `index.html` in your browser**
   ```bash
   open index.html
   # or just double-click the file
   ```

---

## 🛠️ How to Use

1. Enter marks for each subject in the input fields.
2. Use the **+ Add Subject** button to add more subjects as needed.
3. Use the **− Remove** button to delete a subject entry.
4. Click **Calculate Grade** to view:
   - **Total** — Sum of all subject marks
   - **Average** — Mean score across subjects
   - **Grade** — Final letter grade based on the average

## 📐 Grading Scale

| Average Score | Grade |
|---------------|-------|
| 90 – 100      | A+    |
| 80 – 89       | A     |
| 70 – 79       | B     |
| 60 – 69       | C     |
| 50 – 59       | D     |
| Below 50      | F     |

> *Note: The grading scale above is indicative. Refer to the app logic for exact thresholds.*
